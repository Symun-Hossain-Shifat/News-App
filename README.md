# 📰 News App (Next.js + MongoDB + Authentication)

A modern full-stack News Application built with **Next.js**, **MongoDB**, and **Authentication system (Google OAuth / NextAuth)**.  
This is my first full-stack project where I implemented authentication, database integration, and dynamic news features.

---

## 🚀 Live Demo
👉 https://news-app-rust-delta.vercel.app/

---

## 📌 Features

- 🔐 User Authentication (Google Login)
- 🧠 Secure session handling
- 🗄️ MongoDB database integration
- 📰 Dynamic news listing system
- 📱 Fully responsive UI
- ⚡ Fast performance with Next.js
- 🌐 Deployed on Vercel

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Next.js API Routes
- **Authentication:** NextAuth (Google Provider)
- **Database:** MongoDB Atlas
- **Deployment:** Vercel
